# Проэктор

Веб сервис для ведения проектной команды:

- отслеживания действий;
- сбора статистики;
- заполнения характеристик.

## Мотивация

Данный проект создан с целью сбора и анализа ифнормации о действии каждого члена команды, чтобы объективно оценивать вклад в работу.
